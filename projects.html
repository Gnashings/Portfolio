BOOM

Play as a tank and blow stuff up!

This project made HEAVY use of scriptable objects within unity. From the explosions to the abilities of the tank itself. First project I made that utilized object pooling as well, useful when you have like 20 shots flying across the screen.

Ghastly Grove

This game is rather simple, you just need to race to the far-side of the level before Larry the Ghost finds you.

Larry stares you down before chasing, patrols the map, checks the last location he remembered seeing you, gets faster and faster the more often you escape him...

PROJECT-H4S73

This is probably the most mechanically diverse game I had ever programmed. The game features the ability to slow down time, "shifting" between dimensions, semi-random level layouts. Each level had a different gimmick as well!
