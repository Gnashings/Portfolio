About

i'm programming as hard as i can

My name is Nash.

I'm currently in the game Development Program at UCF with a minor in Computer Science. My main focus is programming and technical design.

When it comes to programming in games, I often seek to create quick and simple tools to make for my designers. In many of my projects, especially more recent ones, you'll find a lot of drag and drop scripts for testing mechanics. Keeping things compartmentalized and organized is a big focus for me in any project I work on. I make heavy use of design patterns in my programming! State machines, observer, and command patterns can all be found in my projects.
